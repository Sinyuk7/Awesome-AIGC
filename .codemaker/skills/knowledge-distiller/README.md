# Knowledge Distiller

将多源材料蒸馏为单一结构化认知型知识文档的交互式 Skill。

## 触发条件

当用户提到以下内容时，应该使用这个 Skill：
- "蒸馏"、"distill"
- "整理知识"、"合并文档"
- "提炼知识"、"归纳总结目录内容"
- 指定一个目录，希望将其中的知识整合成一份结构化文档

## 核心特性

1. **交互式对话驱动** — 像 Agent 一样逐步与用户确认蒸馏范围、焦点、模块选择等细节
2. **6 阶段工作流程** — 范围确认 → 蒸馏焦点 → 模块选择 → 输出格式分析 → 执行蒸馏 → 完成报告
3. **渐进式写入** — 每个模块完成后立即写入，防止超时
4. **灵活的意图理解** — 从用户输入理解蒸馏意图，无预设模式
